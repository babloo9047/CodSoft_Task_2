# Student Grade Calculator (Java)

## 📌 Project Overview
The **Student Grade Calculator** is a Java-based application designed to calculate students' academic performance by taking subject marks as input and generating total marks, percentage, grades, and pass/fail status.

This project was developed as part of the **Oasis Infobyte Java Development Internship**.

---

## 🚀 Features
- Add marks for multiple subjects
- Calculate total marks
- Calculate average percentage
- Automatic grade assignment
- Pass/Fail status
- Multiple student support
- Result history
- Class statistics
- Input validation

---

## 🛠 Technologies Used
- Java
- Arrays
- Loops
- Conditional Statements
- Scanner Class
- OOP Concepts

---

## ⚙ Grade Criteria
| Percentage | Grade |
|------------|-------|
| 90+        | A+    |
| 80-89      | A     |
| 70-79      | B     |
| 60-69      | C     |
| 40-59      | D     |
| Below 40   | F     |

---

## ▶ How to Run
1. Compile the Java file  
2. Run the program  
3. Enter student details  
4. Input marks for all subjects  
5. View generated result
